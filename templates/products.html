Import sản phẩm từ Excel

Tải file mẫu, điền dữ liệu (mỗi dòng một sản phẩm), rồi upload để xem trước trước khi import. File .xlsx có nhiều tab (sheet) ở dưới — hệ thống sẽ đọc và gộp tất cả các tab có cùng dòng tiêu đề (SKU, Tên sản phẩm, …).

File mẫu .xlsx File mẫu .csv
Bắt buộc: SKU, Tên sản phẩm.
Danh mục: cột Danh mục cha + Danh mục con, hoặc Danh mục dạng Cha > Con.
Quy đổi hộp/thùng → chiếc: Đơn vị tồn kho = Chiếc, Đơn vị nhập = Hộp (hoặc Thùng), Hệ số quy đổi = 20 (1 hộp = 20 chiếc). Cùng cột dùng cho mét/cuộn (VD: Cuộn, 50).
Ảnh: cột URL hình ảnh (link https://...) — có URL thì hiển thị ảnh online, không có thì dùng ảnh upload trên web.
Export và import dùng cùng định dạng cột — tải lại file mẫu nếu file cũ thiếu cột.
